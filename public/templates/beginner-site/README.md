# Simpele website-template

Deze projectmap werkt zonder installatie.

## Starten

1. Open de map.
2. Dubbelklik op `index.html`.
3. De website opent in je browser.

## Welk bestand verander je?

- `index.html`: tekst en onderdelen zoals navigatie, hero, kaarten, contact en footer.
- `style.css`: kleuren, ruimte, lettergroottes en de mobiele opmaak.
- `script.js`: opent en sluit alleen het menu op een klein scherm.

## Makkelijke eerste aanpassingen

1. Vervang `MijnLogo` en alle voorbeeldteksten in `index.html`.
2. Verander de vijf kleuren bovenaan `style.css`.
3. Open `index.html` opnieuw en controleer iedere wijziging.

Voeg pas nieuwe code toe wanneer deze basis goed werkt.
