01

HTML · basis

Een lege webpagina

Begin hier

Dit is het vaste begin van iedere HTML-pagina.

  1. Maak index.html.
  2. Plak deze code.
  3. Vervang de titel en tekst.
index.html
<!doctype html>
<html lang="nl">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css">
    <title>Mijn website</title>
  </head>
  <body>
    <h1>Hallo wereld!</h1>
    <script src="script.js"></script>
  </body>
</html>
02

HTML · onderdeel

Navigatie

Menu

Een simpele bovenbalk met een logo en drie links.

  1. Plak dit bovenaan in <body>.
  2. Pas de linkteksten aan.
  3. Laat href verwijzen naar een id.
index.html
<header class="site-header">
  <a class="logo" href="#">MijnLogo</a>
  <nav>
    <a href="#home">Home</a>
    <a href="#over">Over mij</a>
    <a href="#contact">Contact</a>
  </nav>
</header>
03

HTML · onderdeel

Hero-sectie

Welkom

Het eerste grote blok dat bezoekers op je homepage zien.

  1. Plak dit onder je navigatie.
  2. Schrijf in één zin wat je maakt.
  3. Laat de knop naar je inhoud wijzen.
index.html
<section class="hero" id="home">
  <p>Welkom op mijn website</p>
  <h1>Ik maak simpele websites.</h1>
  <p>Hier leg ik kort uit wat bezoekers kunnen vinden.</p>
  <a class="button" href="#over">Lees meer</a>
</section>
04

HTML · onderdeel

Drie kaarten

Overzicht

Handig voor diensten, projecten of eigenschappen.

  1. Maak één section.
  2. Gebruik voor elk kaartje een article.
  3. Kopieer een kaart als je er meer wilt.
index.html
<section class="cards" id="over">
  <article><h2>Snel</h2><p>Een korte uitleg.</p></article>
  <article><h2>Simpel</h2><p>Een korte uitleg.</p></article>
  <article><h2>Duidelijk</h2><p>Een korte uitleg.</p></article>
</section>
05

HTML · onderdeel

Contactformulier

Contact

Een duidelijk formulier. Voor echt verzenden is later een backend nodig.

  1. Plak dit bij je contactgedeelte.
  2. Laat ieder label bij een veld staan.
  3. Test of je met Tab door de velden kunt.
index.html
<form id="contact">
  <label for="naam">Naam</label>
  <input id="naam" name="naam" type="text" required>

  <label for="email">E-mail</label>
  <input id="email" name="email" type="email" required>

  <label for="bericht">Bericht</label>
  <textarea id="bericht" name="bericht" rows="5"></textarea>

  <button type="submit">Versturen</button>
</form>
06

HTML · onderdeel

Footer

Onderkant

Het afsluitende blok met copyright en belangrijke links.

  1. Plak dit vlak voor </body>.
  2. Pas de naam en links aan.
  3. Houd de footer kort.
index.html
<footer class="site-footer">
  <p>© 2026 Mijn website</p>
  <nav aria-label="Footer">
    <a href="#home">Naar boven</a>
    <a href="#contact">Contact</a>
  </nav>
</footer>
07

CSS · basis

Rustige basisstijl

Vormgeving

Een nette start voor tekst, kleuren en witruimte.

  1. Maak style.css.
  2. Plak dit bovenaan.
  3. Verander eerst alleen de drie kleuren.
style.css
* { box-sizing: border-box; }

body {
  margin: 0;
  color: #172033;
  background: #fffaf0;
  font-family: Arial, sans-serif;
  line-height: 1.6;
}

section { padding: 80px 24px; }
a { color: #3457d5; }
img { max-width: 100%; }
08

CSS · onderdelen

Stijl voor de blokken

Alles samen

Geeft de navigatie, hero, kaarten, knop en footer een simpele vorm.

  1. Plak dit onder je basisstijl.
  2. Gebruik dezelfde classnamen in HTML.
  3. Verander daarna kleuren en ruimte.
style.css
.site-header, .site-footer {
  display: flex;
  gap: 24px;
  justify-content: space-between;
  padding: 20px 6vw;
  background: #172033;
  color: white;
}

.hero { min-height: 65vh; text-align: center; }
.button { display: inline-block; padding: 12px 20px; background: #ff6b35; color: white; }
.cards { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; }
.cards article { padding: 24px; background: white; border: 1px solid #dfe3ec; }

@media (max-width: 700px) {
  .cards { grid-template-columns: 1fr; }
}
09

JavaScript · klik

Knop verandert tekst

Interactie

De kleinste manier om een klik zichtbaar iets te laten doen.

  1. Geef je knop het id actie-knop.
  2. Plak dit in script.js.
  3. Klik op de knop om te testen.
script.js
const knop = document.querySelector('#actie-knop');

knop.addEventListener('click', () => {
  knop.textContent = 'Gelukt!';
});
10

JavaScript · menu

Mobiel menu openen

Navigatie

Zet een class aan of uit wanneer iemand op de menuknop klikt.

  1. Gebruik de classnamen uit de code.
  2. Plak dit in script.js.
  3. Maak .is-open zichtbaar in CSS.
script.js
const menuKnop = document.querySelector('.menu-knop');
const menu = document.querySelector('.menu');

menuKnop.addEventListener('click', () => {
  menu.classList.toggle('is-open');
});
Geen simpele snippet gevonden.

Probeer een kort woord zoals “menu”, “kaart”, “kleur”, “knop” of “contact”.
