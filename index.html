      Hoofdstuk 03

      Zet XAMPP één keer goed klaar.

      XAMPP levert Apache, PHP én MySQL. Voor de SQLite-route start je alleen Apache. Voor de MySQL-route start je Apache en MySQL. Kies één database per project.

      1. 01

        Start de onderdelen voor jouw route

        Open het XAMPP Control Panel. Start altijd Apache. Start bij PHP + MySQL ook MySQL. Beide rijen moeten groen worden.

      2. 02

        Controleer de juiste projectmap

        Je projecten staan normaal in C:\xampp\htdocs\. Maak daar straks de map studenten-crud.

      3. 03

        Controleer SQLite

        Open XAMPP → Apache → Config → PHP (php.ini). Zoek de volgende regels en haal de puntkomma vooraan weg.

        php.ini
        extension=pdo_sqlite
extension=sqlite3

        Daarna: sla php.ini op en herstart Apache. Zonder herstart ziet PHP de wijziging niet.

      4. 04

        Doe de 10-seconden test

        Maak tijdelijk test.php in htdocs met onderstaande code. Zie je SQLite werkt? Dan kun je verder.

        <?php echo extension_loaded('pdo_sqlite') ? 'SQLite werkt' : 'SQLite ontbreekt';
      !
      SQLite en MySQL zijn alternatieven.

      SQLite is één bestand en het makkelijkst. MySQL draait als XAMPP-service en kun je bekijken via http://localhost/phpmyadmin/.

        Belangrijk begrip

        PDO is de standaard PHP-laag waarmee je databasevragen uitvoert. Een prepared statement houdt gebruikersinvoer apart van SQL en voorkomt SQL-injectie.

        config/database.php · kern
        $connection = new PDO('sqlite:' . $databaseFile, null, null, [
    PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
]);

$connection->exec('PRAGMA foreign_keys = ON');
$connection->exec(file_get_contents($schemaFile));

        Regel voor regel

        new PDO('sqlite:' ...)
        Opent SQLite op het opgegeven bestandspad.
        ERRMODE_EXCEPTION
        Zorgt dat een SQL-fout een duidelijke foutmelding wordt.
        foreign_keys = ON
        Dwingt de relatie tussen student en land echt af.
        schema.sql
        Maakt tabellen alleen als ze nog niet bestaan.

        03

        Update — één rij wijzigen

        Laad eerst de bestaande student in het formulier. Bij opslaan bepaalt de verborgen id welke rij verandert.

        index.php · UPDATE · compleet vervangblok
        case 'update_student':
    $id = positive_id($_POST['id'] ?? null, 'Student-ID');
    $student = student_input($db);
    $student['id'] = $id;
    $statement = $db->prepare(
        'UPDATE studenten SET
            studentnummer = :studentnummer,
            voornaam = :voornaam,
            achternaam = :achternaam,
            email = :email,
            opleiding = :opleiding,
            land_id = :land_id,
            status = :status,
            updated_at = CURRENT_TIMESTAMP
         WHERE id = :id'
    );
    $statement->execute($student);

    if ($statement->rowCount() === 0) {
        $exists = $db->prepare('SELECT COUNT(*) FROM studenten WHERE id = ?');
        $exists->execute([$id]);
        if ((int) $exists->fetchColumn() === 0) {
            throw new RuntimeException('Student niet gevonden.');
        }
    }

    flash('success', 'Student is bijgewerkt.');
    break;
        Waarom WHERE?

        Zonder WHERE id = :id wijzig je iedere student.

        Waarom verborgen ID?

        Het formulier weet zo welke rij je aan het bewerken bent.

        Zelfde validatie?

        Ja. Create en Update gebruiken allebei student_input().

        04

        Delete — veilig verwijderen

        Verwijderen gebruikt ook POST, een CSRF-token en een exact ID. Gebruik hiervoor nooit een simpele GET-link.

        index.php · DELETE · compleet vervangblok
        case 'delete_student':
    $id = positive_id($_POST['id'] ?? null, 'Student-ID');
    $statement = $db->prepare('DELETE FROM studenten WHERE id = ?');
    $statement->execute([$id]);

    if ($statement->rowCount() === 0) {
        throw new RuntimeException('Student niet gevonden.');
    }

    flash('success', 'Student is verwijderd.');
    break;

        Waarom ??

        Dat is een positionele placeholder voor het ID.

        Waarom rowCount?

        Je ontdekt daarmee of er echt een rij is verwijderd.

        $intent = post_text('intent');

        if ($intent === 'delete_relation') {
            $id = positive_id($_POST['id'] ?? null, 'Student-ID');
            $statement = $db->prepare('DELETE FROM relatie_demo_studenten WHERE id = ?');
            $statement->execute([$id]);
            if ($statement->rowCount() === 0) {
                throw new RuntimeException('Student niet gevonden.');
            }
        } else {
            $naam = required_text('naam', 'Naam', 80);
            $landId = positive_id($_POST['land_id'] ?? null, 'Land');
            $landCheck = $db->prepare('SELECT COUNT(*) FROM landen WHERE id = ?');
            $landCheck->execute([$landId]);
            if ((int) $landCheck->fetchColumn() !== 1) {
                throw new RuntimeException('Kies een bestaand land.');
            }

            if ($intent === 'create_relation') {
                $statement = $db->prepare(
                    'INSERT INTO relatie_demo_studenten (naam, land_id) VALUES (:naam, :land_id)'
                );
                $statement->execute(['naam' => $naam, 'land_id' => $landId]);
            } else {
                $id = positive_id($_POST['id'] ?? null, 'Student-ID');
                $statement = $db->prepare(
                    'UPDATE relatie_demo_studenten SET naam = :naam, land_id = :land_id WHERE id = :id'
                );
                $statement->execute(['naam' => $naam, 'land_id' => $landId, 'id' => $id]);
                if ($statement->rowCount() === 0) {
                    $exists = $db->prepare('SELECT COUNT(*) FROM relatie_demo_studenten WHERE id = ?');

                    if ((int) $exists->fetchColumn() === 0) {

declare(strict_types=1);

function database(): PDO
{
    static $db;
    if ($db instanceof PDO) return $db;

    $path = dirname(__DIR__) . '/data/bookings.sqlite';
    $db = new PDO('sqlite:' . $path, null, null, [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
        PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
        PDO::ATTR_EMULATE_PREPARES => false,
    ]);
    $db->exec('PRAGMA foreign_keys = ON');
    $db->exec('PRAGMA busy_timeout = 5000');
    return $db;
}

function requiredText(string $key, int $max = 190): string
{
    $value = trim((string) ($_POST[$key] ?? ''));
    if ($value === '' || mb_strlen($value) > $max) {
        throw new InvalidArgumentException("Ongeldige waarde voor {$key}.");
    }
    return $value;
}

function positiveId(string $key): int
{
    $id = filter_input(INPUT_POST, $key, FILTER_VALIDATE_INT,
        ['options' => ['min_range' => 1]]);
    if ($id === false || $id === null) throw new InvalidArgumentException("Ongeldig {$key}.");
    return (int) $id;
}

function localDateTime(string $key, DateTimeZone $zone): DateTimeImmutable
{
    $date = DateTimeImmutable::createFromFormat('!Y-m-d\TH:i', requiredText($key), $zone);
    if (!$date) throw new InvalidArgumentException('Ongeldige datum of tijd.');
    return $date;
}

{

$bulkIntents = ['bulk_activate', 'bulk_deactivate', 'bulk_delete'];
if ($_SERVER['REQUEST_METHOD'] === 'POST'
    && in_array(post_text('intent'), $bulkIntents, true)
) {
    verify_csrf($_POST['_token'] ?? null);
    $ids = array_values(array_unique(array_filter(
        array_map('intval', $_POST['selected_ids'] ?? []),
        fn (int $id): bool => $id > 0
    )));
    if ($ids === [] || count($ids) > 100) {
        throw new RuntimeException('Kies tussen één en honderd geldige rijen.');
    }

    $placeholders = implode(',', array_fill(0, count($ids), '?'));
    $intent = post_text('intent');
    if ($intent === 'bulk_delete') {
        $sql = "DELETE FROM studenten WHERE id IN ($placeholders)";
        $parameters = $ids;
    } else {
        $status = $intent === 'bulk_activate' ? 'actief' : 'inactief';
        $sql = "UPDATE studenten SET status = ? WHERE id IN ($placeholders)";
        $parameters = [$status, ...$ids];
    }
    $db->prepare($sql)->execute($parameters);
    redirect('index.php');
}
